# 📊 Retail Sales Analyzer

A comprehensive Python tool for analyzing retail sales data, providing key business insights through statistical analysis and interactive visualizations.

## 🎯 Features

### Data Processing
- **Automated Data Loading** - Seamlessly imports CSV sales data
- **Intelligent Data Cleaning** - Automatically removes rows with missing values

### Key Metrics
- **Total Sales per Product** - Understand product-wise revenue distribution
- **Best-Selling Product** - Identify your top performer instantly
- **Average Daily Sales** - Track daily revenue benchmarks

### Visualizations
- **Sales Trends Over Time** - Line chart showing temporal sales patterns
- **Product Performance** - Bar chart comparing product sales
